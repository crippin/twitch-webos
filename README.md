## Install

```bash
$ git clone https://github.com/crippin/twitch-webos.git
$ cd twitch-webos
$ yarn
```
## Run in browser

```bash
$ yarn start
```

## Build bundle.js

```bash
$ yarn build-prod
```
## Build ipk

```bash
$ yarn build-ipk
```

## Install to tv
```bash
$ ares-isntall app -d tv-name
```
